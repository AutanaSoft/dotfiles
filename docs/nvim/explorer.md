# Explorer

Press `Space e` to open the file explorer (provided by `snacks.nvim`). Press it again or `q` to close.

## Navigation

| Action | Shortcut |
| --- | --- |
| Move up / down | `j` / `k` |
| Open file or expand folder | `l` or `Enter` |
| Collapse folder or go to parent | `h` |
| Go to parent directory | `Backspace` |
| Jump to current file | `.` |
| Close all folders | `Z` |
| Toggle hidden files | `H` |
| Toggle ignored files (gitignore) | `I` |
| Toggle preview pane | `P` |
| Refresh tree | `u` |
| Open with system application | `o` |

## File Operations

| Action | Shortcut |
| --- | --- |
| Add file or folder | `a` |
| Rename | `r` |
| Delete | `d` |
| Move selected files | `m` |
| Copy selected files | `c` |
| Yank (copy) file path | `y` |
| Paste yanked file | `p` |

**Selection-based workflow:** Select multiple files with `Tab`, then press `m` to move them all or `c` to copy them all to the current directory.
