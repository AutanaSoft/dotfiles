# Zellij Plugins

This directory contains the Zellij plugin binaries used by `config/zellij/config.kdl`.

## Versioned Binaries

- `zellij_forgot.wasm` — shortcut helper opened with `Alt y`
- `zjframes.wasm` — pane frame plugin
- `zjstatus.wasm` — status bar plugin

Copying `config/zellij/` into `~/.config/zellij/` installs these binaries together with the main
config.
