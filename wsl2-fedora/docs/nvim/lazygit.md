# Lazygit

Press `Space g g` to open Lazygit in a floating terminal. Use `q` to close it.

## Common Actions

| Action | Shortcut |
| --- | --- |
| Stage / unstage file | `Space` |
| Stage all | `a` |
| Discard changes | `d` |
| Commit staged | `c` |
| Commit with editor | `C` (`Shift+c`) |
| Amend last commit | `A` (`Shift+a`) |
| Push | `P` (`Shift+p`) |
| Pull | `p` |
| Switch branch | `b` |
| Create branch | `n` |
| View keybindings | `?` |
| Quit Lazygit | `q` or `Esc` |
